\section{概要}
レポート課題として課題Aを選択し, 3層フィードフォワード型の
ニューラルネットとバックプロパゲーション学習をシミュレーション
するプログラムを作成し, 識別実験を行った. 

識別実験に利用したデータ群は
The MNIST database of handwritten digits
であり, 過去に様々な分類器において識別能力を図るために利用されている. 
\cite{mnist}
